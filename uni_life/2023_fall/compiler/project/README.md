# Mini Rust

## Support
- [x] `println!` macros to standard output
    - [ ] expression: `"{}"`  
    - [x] escape characters: `\n`, `\t` 
- [x] `let` keyword
- [x] `u8`, `str` type keyword (no type checking)
- [ ] `fn` for function declaration

## Example
- input
```rust
let a: u8 = 100;
let b: u8 = 200;
let c: u8 = a + b;

println!("result: {}", c);
```

- output
```bash
result: 300
```
